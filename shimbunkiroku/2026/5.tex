    0502联合早报·四川成都发生连环撞人事故 致一死11伤
} {
    \setlength{\parindent}{2em}
    （成都综合讯）中国四川省成都市发生一起严重交通事故，一名轿车司机与行人发生碰撞后逃逸，途中又接连撞上多辆车及行人，造成一人死亡、11人受伤。司机当场被捕，警方已排除司机酒驾和毒驾嫌疑。

这起事故发生在星期五（5月1日）下午5时20分许，地点在成都市高新区剑南大道。

网传视频显示，一辆在内侧车道等红灯的粉银色轿车，在绿灯亮起后突然加速右转，冲入人行横道，撞倒多名正在过马路的行人，至少六人倒地不起。随后，这辆轿车冲上人行道，又倒车驶回马路，并再次撞上一辆载有两人的电单车，其中一人被撞飞倒地，另一人连同电单车被顶推前行。

成都市公安局交通管理局当晚通报称，经初步调查，31岁的李姓男子驾驶一辆小型轿车行驶至剑南大道与天府四街交叉路口右转时，与行人发生碰撞。他随后驾车逃离现场，在逃逸过程中先后与其他车辆及行人发生碰撞。事故共造成一人死亡，11人不同程度受伤；目前所有伤者均已送医治疗。

案发后，李姓男子涉嫌交通肇事逃逸被当场抓获，经检测，已排除他酒驾、毒驾嫌疑。目前，事故正在进一步调查处理中。

警方还在通告中呼吁公众勿传播现场图片、视频及“相关不实信息”，避免对逝者家属及伤者造成二次伤害。
